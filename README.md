# Detecção de Anomalias em Transações Financeiras

Projeto de Machine Learning desenvolvido para a identificação de fraudes em transações financeiras.

## 🛠️ Tecnologias Utilizadas
* Python
* Scikit-Learn (Isolation Forest)
* Pandas
* Pytest (Testes automatizados)

## 🚀 Como rodar o projeto
1. Clone o repositório.
2. Instale as dependências: `pip install -r requirements.txt`
3. Execute os testes: `python -m pytest`
4. Execute o modelo: `python main.py`

## 📊 Sobre o Modelo
Este projeto utiliza o algoritmo *Isolation Forest* para detectar padrões anômalos. O objetivo é equilibrar a precisão na detecção de fraudes, garantindo confiabilidade no monitoramento de transações.
